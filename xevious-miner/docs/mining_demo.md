```
初始难度：0x00FF (255)

- 难度增加 0x0010 (16)
- 确保挖矿进度
- 防止无限循环

难度变化示例:
尝试 0-9999:    难度 = 0x00FF (255)
尝试 10000-19999: 难度 = 0x010F (271)
尝试 20000-29999: 难度 = 0x011F (287)
...
最大难度：0xFFFF (65535)
```

```

- 语言：Python (解释型)

实际 Z80 @ 3.072 MHz:
- 估算哈希率：~500-1000 H/s

- 语言：汇编 (原生)

(主要由于 Python 运行在现代硬件上)
```

## 技术细节

### 哈希函数实现

```python
def pseudo_hash(nonce, seed=0x1234):
    """
    简化的伪哈希函数
    模拟 Z80 可执行的计算
    """
    # XOR with seed
    value = nonce ^ seed
    
    # Linear congruential generator step
    value = ((value * 1103515245) + 12345) & 0x7FFFFFFF
    
    # Mix bits
    value = value ^ (value >> 16)
    
    # Return 16-bit result
    return value & 0xFFFF
```

### 难度检查

```python
def check_difficulty(hash_value, difficulty=0x00FF):
    """
    检查哈希值是否满足难度要求
    """
    return hash_value <= difficulty

# 示例:
# hash = 0x00AB (171)
# difficulty = 0x00FF (255)
# 171 <= 255 → True ✅ 区块有效!

# hash = 0x01AB (427)
# difficulty = 0x00FF (255)
# 427 <= 255 → False ❌ 需要继续挖矿
```
