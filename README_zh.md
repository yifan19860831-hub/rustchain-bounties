## 什么是 RTC？

**RTC (RustChain Token)** 是 [RustChain](https://github.com/Scottcjn/Rustchain) 的原生加密货币。RustChain 是一个"古老硬件挖矿收益更高"的区块链。RTC 参考价：**0.10 美元**。

完成验证后，RTC 会发送到你的钱包地址。

## 如何赚取

### 1. 选择赏金
浏览 [开放赏金](https://github.com/Scottcjn/rustchain-bounties/issues?q=is%3Aissue+is%3Aopen+label%3Abounty)，找适合自己的。

| 难度 | 标签 | 典型奖励 |
|-----------|-------|---------------|
| 新手 | `good first issue` | 1-5 RTC |
| 普通 | `standard` | 5-25 RTC |
| 重要 | `major` | 25-100 RTC |
| 关键 | `critical`, `red-team` | 100-200 RTC |

### 2. 认领
在 Issue 下留言：**"我想做这个"**

### 3. 提交作品
- **代码赏金**: 提交 PR 并在 Issue 中关联
- **内容赏金**: 发布内容并在 Issue 中关联

### 4. 收款
验证通过后，RTC 发送到钱包。首次？我们会帮你设置。
