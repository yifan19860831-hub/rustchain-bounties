鬠ｿFerranti Mark 1* Paper Tape Programs
=====================================

This directory contains paper tape programs for the Ferranti Mark 1* miner.

Program Format:
- Each line represents one 20-bit word
- Format: OPCODE (5 bits) | ADDRESS (10 bits) | B-LINE (3 bits) | UNUSED (2 bits)
- Octal notation for compatibility with 1950s programming practices

Opcode Values (Octal):
00 - STOP       10 - AND        20 - XOR (extended)
01 - LOAD       11 - OR         21 - AND mask (extended)
02 - STORE      12 - NOT        22 - Compare (extended)
03 - ADD        13 - SHIFT_L    30 - DRUM_READ (extended)
04 - SUB        14 - SHIFT_R    31 - DRUM_WRITE (extended)
05 - MUL        15 - OUTPUT     40 - LOAD_B
06 - DIV        16 - HOOT       41-43 - B-line ops
07 - JUMP       17 - TELE_PRINT
08 - JNEG       20 - RAND
09 - JZER       21 - INPUT

B-Line Encoding (bits 15-13):
000 - No modification
001 - B1
010 - B2
011 - B3
100 - B4
101 - B5
110 - B6
111 - B7

================================================================================
Usage Notes
================================================================================

1. Load programs via paper tape reader (simulated)
2. Programs start at address 000 by default
3. Data area typically starts at address 050
4. All values in octal for historical accuracy
5. B-line 0 always reads as 0 (hardwired)

To run a program:
1. Load tape into simulator
2. Set program counter to start address
3. Execute until STOP instruction
4. Check results in data area
