# ENIAC 架构详细文档

## 1. 概述

ENIAC (Electronic Numerical Integrator and Computer) 是世界上第一台通用电子计算机，1945 年完成于宾夕法尼亚大学摩尔电气工程学院。

## 2. 核心技术规格

### 2.1 数字表示

**字长**: 10 位十进制数字 + 符号位

```
格式：[符号][d9][d8][d7][d6][d5][d4][d3][d2][d1][d0]
范围：-9,999,999,999 到 +9,999,999,999
```

**负数表示**: 10 的补码 (Ten's Complement)

```
正数：直接表示
  +123 = 0000000123

负数：10 的补码
  -1 = 9999999999 (10^10 - 1)
  -123 = 9999999877 (10^10 - 123)
```

### 2.2 累加器 (Accumulators)

ENIAC 有 **20 个累加器**，每个累加器：
- 存储 1 个 10 位十进制数
- 可执行加法和减法
- 通过环形计数器 (ring counter) 实现

**环形计数器实现**:
```
每个十进制位需要：
- 10 个双三极管 (flip-flops)
- 36 个真空管总计

工作原理：
- 接收脉冲"计数"
- 环绕时产生进位脉冲
- 电子模拟机械加法机的数字轮
```

### 2.3 运算速度

| 操作 | 周期数 | 时间 (微秒) | 每秒次数 |
|------|--------|-------------|----------|
| 加法/减法 | 1 | 200 | 5,000 |
| 乘法 (10 位×10 位) | 14 | 2,800 | 357 |
| 除法 (10 位商) | 143 | 28,600 | 35 |
| 平方根 (10 位) | 143 | 28,600 | 3 |

**基本机器周期**: 200 微秒 (100kHz 时钟的 20 个周期)

### 2.4 功能单元

```
ENIAC 模块组成 (共 40 个面板):

累加器 (Accumulators) ........... 20 个
  ├─ 加/减运算
  ├─ 数据存储
  └─ 可并行操作

乘法器 (Multiplier) ............. 1 个
  ├─ 控制 4 个累加器
  └─ 385 次乘法/秒

除法器/平方根器 ................. 1 个
  ├─ 控制 5 个累加器
  ├─ 40 次除法/秒
  └─ 3 次平方根/秒

主程序器 (Master Programmer) .... 1 个
  ├─ 循环序列控制
  └─ 分支逻辑

启动单元 (Initiating Unit) ...... 1 个
  └─ 启动/停止机器

循环单元 (Cycling Unit) ......... 1 个
  └─ 同步所有单元

读取器 (Reader) ................. 1 个
  └─ 控制 IBM 读卡机

打印机 (Printer) ................ 1 个
  └─ 控制 IBM 打卡机

常数发送器 (Constant Transmitter)  1 个
  └─ 存储常量

函数表 (Function Tables) ........ 3 个
  ├─ 每个含 1200 个十路开关
  └─ 查找表功能
```

## 3. 编程模型

### 3.1 插线板编程

ENIAC 不是存储程序计算机。程序通过物理方式设置：

```
编程步骤:
1. 设置插线板连接 (功能单元之间的信号路由)
2. 配置函数表开关 (1200 个开关/表)
3. 设置常数发送器
4. 启动机器

重新编程时间：约 2-3 天
```

### 3.2 程序示例：加法循环

```
目标：计算 1+2+3+...+100

ENIAC 设置:
1. 累加器 0: 初始化为 0 (总和)
2. 累加器 1: 初始化为 1 (计数器)
3. 累加器 2: 存储常量 100
4. 累加器 3: 存储常量 1

程序流程:
[启动] → [累加器 1 加到累加器 0] → [累加器 3 加到累加器 1]
    ↑                                                        ↓
    └──── [比较累加器 1 和 2] ← [如果<100 则继续] ←──────────┘
```

### 3.3 分支能力

ENIAC 可以根据计算结果的符号触发不同操作：

```
if (累加器结果 >= 0):
    执行路径 A
else:
    执行路径 B
```

这通过主程序器和累加器的符号输出实现。

## 4. 物理实现

### 4.1 组件统计

```
真空管 (Vacuum Tubes) ......... 18,000 个
晶体二极管 (Crystal Diodes) ... 7,200 个
继电器 (Relays) ............... 6,000 个
电阻 (Resistors) .............. 70,000 个
电容 (Capacitors) ............. 10,000 个
手工焊接点 .................... ~5,000,000 个
```

### 4.2 真空管类型

```
6SN7 ....... 双三极管，用于累加器 flip-flops
6L7 ........ 逻辑功能
6SJ7 ....... 逻辑功能
6SA7 ....... 逻辑功能
6AC7 ....... 逻辑功能
6L6 ........ 线路驱动器 (面板间脉冲传输)
6V6 ........ 线路驱动器
```

### 4.3 可靠性

```
早期：几乎每天烧坏几个真空管，50% 时间不可用
改进后：约每 2 天烧坏 1 个真空管，15 分钟内可定位

1954 年记录：连续运行 116 小时无故障 (~5 天)
```

### 4.4 物理规格

```
重量：> 30 短吨 (27 吨)
尺寸：100 英尺长 × 10 英尺高 × 3 英尺深
      (30m × 3m × 1m)
面积：300 平方英尺 (28 m²)
功耗：150 kW
```

## 5. 输入/输出

### 5.1 输入

- **IBM 读卡机**: 穿孔卡片输入
- **函数表开关**: 手动设置
- **插线板**: 物理连接配置

### 5.2 输出

- **IBM 打卡机**: 穿孔卡片输出
- **离线打印**: 使用 IBM 会计机 (如 IBM 405)

### 5.3 存储扩展 (1953 年)

```
1953 年添加：
- 100 字磁芯存储器 (Burroughs Corporation)
- 这是 ENIAC 唯一的内部存储扩展
```

## 6. 与 RustChain 挖矿的关联

### 6.1 挑战

ENIAC 的原始架构不适合现代加密挖矿：
- 无二进制运算
- 无位操作指令
- 程序需要物理重配置
- 无网络能力

### 6.2 解决方案

**Python 模拟器方法**:
1. 精确模拟 ENIAC 的十进制累加器
2. 模拟 ENIAC 的运算速度和限制
3. 在模拟器上运行"ENIAC 程序"
4. 通过现代网络提交证明

**硬件指纹**:
- 模拟 ENIAC 真空管热噪声
- 模拟累加器进位延迟
- 模拟函数表开关位置熵

### 6.3 挖矿算法适配

```
RustChain 工作量证明简化版:
1. 取区块头数据
2. 转换为十进制数字序列
3. 使用 ENIAC 累加器执行哈希模拟
4. 检查结果是否满足难度目标
5. 提交证明 (attestation)
```

## 7. 历史意义

ENIAC 是计算机历史上的里程碑：
- ✅ 第一台可编程计算机
- ✅ 第一台全电子计算机
- ✅ 第一台通用数字计算机
- ✅ 图灵完备
- ✅ 能够解决"大量数值问题"

**影响**:
- 奠定了现代计算机基础
- 证明了电子计算的可行性
- 开启了计算机革命
