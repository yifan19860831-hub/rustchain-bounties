## IBM 704 架构规格

### 核心特性

| 特性 | 规格 |
|------|------|
| **推出年份** | 1954 |
| **字长** | 36 位 |
| **内存** | 4,096 字磁芯内存 (18,432 字节) |
| **技术** | 真空管逻辑 |
| **性能** | 12,000 次浮点加法/秒 |
| **输入** | 打孔卡片 (IBM 711) |
| **输出** | 打印机 (IBM 716) / CRT 显示 (IBM 740) |
| **重量** | 19,466 磅 (8.8 吨) |
| **可靠性** | 平均 8 小时故障一次 |

### 寄存器组织

```
┌─────────────────────────────────────────────────────────────┐
│                    IBM 704 寄存器布局                        │
├─────────────────────────────────────────────────────────────┤
│  AC (累加器)      : 38 位 (含符号位)                          │
│  MQ (乘除寄存器)   : 36 位                                    │
│  XR1/XR2/XR4      : 3 个 15 位变址寄存器                       │
│  IC (指令计数器)   : 15 位                                    │
│  SI (感觉指示器)   : 36 位状态标志                            │
└─────────────────────────────────────────────────────────────┘
```

### 指令格式

**Type A** (条件跳转):
```
┌────────┬────────────────┬─────────┬────────────────┐
│ 3 位码  │   15 位减量     │ 3 位标签 │   15 位地址     │
│ prefix │  decrement     │  tag    │   address      │
└────────┴────────────────┴─────────┴────────────────┘
```

**Type B** (大多数指令):
```
┌────────────┬──────┬──────┬─────┬────────────────┐
│ 12 位操作码  │ 2 位标志│ 4 位  │ 3 位 │   15 位地址     │
│   opcode   │ flag │ unused│ tag │   address      │
└────────────┴──────┴──────┴─────┴────────────────┘
```

### 数据格式

**单精度浮点数** (36 位):
```
┌────┬────────────────┬─────────────────────────────┐
│ 1 位│   8 位指数      │      27 位尾数              │
│符号 │  (excess-128)  │    (无隐藏位)               │
└────┴────────────────┴─────────────────────────────┘
```

**定点数**: 二进制符号/幅值格式

**字符**: 6 位 BCD，每字存储 6 个字符

## 移植方案

### 设计原则

由于 IBM 704 是 1954 年的真空管计算机，无法直接运行现代代码，本移植采用**概念验证模拟器**方式：

1. **IBM 704 架构模拟器** - Python 实现完整的 36 位 CPU 模拟
2. **汇编级矿工** - 用 IBM 704 SAP 汇编语言编写核心挖矿逻辑
3. **硬件指纹模拟** - 模拟 1954 年硬件特征（真空管老化、磁芯内存延迟等）
4. **历史文档** - 详细的技术文档和时代背景说明

