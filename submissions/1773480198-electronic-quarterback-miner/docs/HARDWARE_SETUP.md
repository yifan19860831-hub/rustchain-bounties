## 🔌 电路设计

### 整体架构

```
Electronic Quarterback (1978)
         │
         ├──────────────┐
         │              │
    [LED 阵列]      [按钮矩阵]
         │              │
    光敏电阻        NPN 晶体管
         │              │
    Pico ADC       Pico GPIO
         │              │
         └──────┬───────┘
                │
           USB 串口
                │
                ▼
           PC 运行矿工
```

### 按钮模拟电路

每个按钮使用一个 NPN 晶体管模拟按下：

```
Pico GPIO ──┬── 1kΩ ──┬── 基极 (2N2222)
            │         │
           GND       GND

EQ 按钮 1 ──┬── 集电极 (2N2222)
            │
EQ 按钮 2 ──┴── 发射极 (2N2222)
```

当 Pico GPIO 输出 HIGH 时，晶体管导通，模拟按钮按下。

### LED 传感电路

使用光敏电阻读取 LED 状态：

```
+5V ──┬── 光敏电阻 ──┬── Pico ADC
      │              │
      │             10kΩ
      │              │
     GND ────────────┴── GND
```

LED 亮起时，光敏电阻阻值下降，ADC 读数上升。

## 📐 PCB 布局建议

### 尺寸约束
Electronic Quarterback 外壳内部空间有限，建议：
- PCB 尺寸：≤ 30mm × 20mm
- 使用薄型元件
- 光敏电阻用导线延长，灵活定位

### 布局要点
1. **光敏电阻位置**: 对准 EQ 的 LED 阵列，距离 5-10mm
2. **晶体管排列**: 靠近 EQ 按钮触点，减少走线
3. **USB 接口**: 外壳开孔引出，或改用导线延长

## 🔧 安装步骤

### 步骤 1: 准备 Electronic Quarterback

⚠️ **警告**: 此过程需要拆开设备，可能损坏外壳。

1. 移除电池盖和电池
2. 卸下外壳螺丝（通常 4-6 颗）
3. 小心分离前后壳（注意塑料卡扣）
4. 露出 PCB 和 LED 阵列

### 步骤 2: 焊接按钮连接

1. 识别按钮触点（通常有两个焊盘）
2. 焊接细导线到每个按钮的两个触点
3. 导线另一端连接到 Pico 桥接板的晶体管

**按钮映射**:
```
EQ 按钮    →  Pico GPIO
─────────────────────
Up         →  GPIO 2
Down       →  GPIO 3
Left       →  GPIO 4
Right      →  GPIO 5
Pass       →  GPIO 6
Reset      →  GPIO 7
```

### 步骤 3: 安装光敏传感器

1. 将光敏电阻定位在 LED 阵列前方 5-10mm
2. 使用热熔胶固定（可移除）
3. 避免遮挡用户视线（如果需要保留显示功能）

**传感器位置**:
- 传感器 1: 对准 QB 位置 LED
- 传感器 2: 对准码数/状态 LED

### 步骤 4: 供电方案

**选项 A: 使用 EQ 原装电池**
- 从 EQ PCB 的 VCC/GND 焊盘引线
- 电压：9V → 需要稳压到 5V（使用 AMS1117-5.0）

**选项 B: 独立 USB 供电**
- Pico 通过 USB 供电
- EQ 使用原装电池
- 仅信号连接（共地）

推荐**选项 B**，更简单且隔离更好。

### 步骤 5: 外壳改装

1. 在外壳侧面开孔引出 USB 线
2. 或使用延长线将 Pico 放在外部
3. 重新组装外壳，确保不挤压导线

## 🔍 故障排除

### 问题 1: 按钮无响应
**症状**: EQ 不响应按钮模拟

**解决**:
1. 检查晶体管极性（E/C 是否接反）
2. 测量基极电压（应为 3.3V when HIGH）
3. 检查按钮触点焊接是否良好

### 问题 2: LED 读数无变化
**症状**: ADC 值恒定，不随 LED 状态变化

**解决**:
1. 调整光敏电阻位置（更靠近 LED）
2. 检查环境光干扰（遮光处理）
3. 增加上拉/下拉电阻值

### 问题 3: 时序方差不够
**症状**: 响应时间过于均匀（疑似模拟）

**解决**:
1. 确保使用真实 EQ 硬件
2. 检查光敏电阻响应速度
3. 增加采样次数（12 → 24）

## 🎯 性能优化

### 提高信噪比
1. **遮光处理**: 用黑色胶带包裹传感器区域
2. **滤波**: 软件上取多次读数平均
3. **同步**: 在按钮按下后固定延迟读取

### 降低延迟
1. **提高串口波特率**: 115200 → 921600
2. **减少采样次数**: 12 → 8（略降低精度）
3. **优化固件**: 使用中断代替轮询

## 📋 安全注意事项

1. **静电防护**: 操作前触摸接地金属
2. **电池安全**: 断开电池后再焊接
3. **温度控制**: 烙铁温度 ≤ 350°C，避免损坏 PCB
4. **可逆性**: 尽量使用可移除的固定方式（热熔胶 vs 强力胶）
