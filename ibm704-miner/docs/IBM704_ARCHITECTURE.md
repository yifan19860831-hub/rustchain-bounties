# IBM 704 架构详解

## 历史背景

**IBM 704** 是 IBM 于 1954 年推出的大型电子数字计算机，由 **John Backus** 和 **Gene Amdahl** 设计。它是：

- ✅ **第一台量产的浮点运算硬件计算机**
- ✅ **第一台使用磁芯内存的计算机**
- ✅ **FORTRAN 和 LISP 的诞生平台**
- ✅ **IBM 700/7000 系列科学架构的基础**

### 关键历史成就

| 年份 | 成就 |
|------|------|
| 1954 | IBM 704 发布 |
| 1954 | MUSIC-N（第一台计算机音乐程序） |
| 1955 | FORTRAN 开发开始 |
| 1956 | LISP 概念形成 |
| 1957 | 用于追踪 Sputnik 卫星轨道 |
| 1957 | Perceptron（第一个神经网络）实现 |

### 变址寄存器特性

IBM 704 有 **3 个变址寄存器**（XR1、XR2、XR4），注意编号是 1、2、4 而非 1、2、3！

```python
# 变址寄存器使用示例
# 指令中的 3 位 tag 字段指定使用哪个寄存器

tag = 0b001  # 使用 XR1
tag = 0b010  # 使用 XR2
tag = 0b100  # 使用 XR4
tag = 0b011  # XR1 OR XR2 (按位或，不是相加!)
tag = 0b111  # XR1 OR XR2 OR XR4

# IBM 704 的变址是"减量"模式
# 有效地址 = 指令地址 - 变址寄存器内容
effective_address = instruction_address - XR_value
```

## 指令格式

### Type A 指令（条件跳转）

```
┌─────────┬──────────────────┬─────────┬──────────────────┐
│ 3 位前缀  │   15 位减量字段    │ 3 位标签 │   15 位地址字段    │
│ (33-35) │    (18-32)       │ (15-17) │     (0-14)       │
└─────────┴──────────────────┴─────────┴──────────────────┘
  bits:     3 bits          15 bits    3 bits      15 bits

用途：条件跳转，基于变址寄存器的值
```

**Type A 操作码**（前缀）：
- `101` = STR (Store Register) - IBM 709 才实现
- `110` = SXD (Set Index Decrement)
- `111` = LXD (Load Index with Decrement)
- 其他 = 条件跳转

### Type B 指令（大多数指令）

```
┌──────────────┬──────┬──────┬─────┬──────────────────┐
│ 12 位操作码   │ 2 位  │ 4 位  │ 3 位 │   15 位地址字段    │
│   (24-35)    │标志  │ 未用 │ 标签 │     (0-14)       │
│              │(22-23)│(18-21)│(15-17)│                  │
└──────────────┴──────┴──────┴─────┴──────────────────┘
  bits:    12 bits     2 bits  4 bits  3 bits    15 bits

用途：数据传输、算术运算、浮点运算等
```

## 数据格式

### 定点数格式

```
┌────┬──────────────────────────────────────────────────┐
│ S  │                  35 位幅值                       │
│(35)│                      (0-34)                      │
└────┴──────────────────────────────────────────────────┘

符号/幅值表示法（Sign-Magnitude）
范围：-(2^35-1) 到 +(2^35-1)
```

### 单精度浮点数格式

IBM 704 是**第一台硬件支持浮点运算的计算机**！

```
┌────┬────────────────┬─────────────────────────────────┐
│ S  │  8 位指数       │         27 位尾数               │
│(35)│  (27-34)       │            (0-26)               │
│    │  (excess-128)  │        (无隐藏位)                │
└────┴────────────────┴─────────────────────────────────┘

值 = (-1)^S × 0.fraction × 2^(exponent-128)
```

**浮点范围**：
- 最大正数：≈ 10^38
- 最小正数：≈ 10^-38
- 精度：约 8 位十进制数字

**浮点指令**：
- `FAD` - 浮点加法 (320 octal)
- `FSB` - 浮点减法 (321 octal)
- `FMP` - 浮点乘法 (322 octal)
- `FDH` - 浮点除法 (323 octal)
- `DNORM` - 规格化 (302 octal)
- `ZERO` - 测试零 (303 octal)

## 内存系统

### 磁芯内存

```
┌─────────────────────────────────────────────────────────────┐
│  IBM 737 磁芯存储单元                                        │
├─────────────────────────────────────────────────────────────┤
│  容量          : 4,096 字 (36 位每字)                         │
│  总存储        : 147,456 位 (18,432 字节)                     │
│  访问时间      : 12 微秒                                     │
│  周期时间      : 12 微秒                                     │
│  技术          : 铁氧体磁环                                   │
│  功耗          : ~3 kW                                       │
│  尺寸          : 约 1 立方米                                 │
└─────────────────────────────────────────────────────────────┘
```

**磁芯内存原理**：
- 每个磁环存储 1 位（磁化方向表示 0 或 1）
- 4096 个字 × 36 位 = 147,456 个磁环
- 通过导线网格寻址
- **非易失性**：断电后数据保留

### 内存地址

```
地址范围：0 - 4095 (十进制)
         0 - 7777 (八进制)  ← IBM 704 使用八进制！

寻址方式：
- 直接寻址：15 位地址字段
- 变址寻址：地址 - XR 内容
```

## 编程示例

### 简单加法程序

```assembly
# IBM 704 SAP Assembly - 简单加法

        ORG 0           # 程序从地址 0 开始

START   LDA A           # 加载 A 到 AC
        ADD B           # 加 B
        STA RESULT      # 存储结果
        HPR             # 停机

# 数据
A       DEC 100         # A = 100
B       DEC 200         # B = 200
RESULT  DEC 0           # 结果存储位置

        END START
```

### 浮点运算示例

```assembly
# IBM 704 浮点乘法

        ORG 0

START   LDA MULTIPLICAND  # 加载被乘数
        FMP MULTIPLIER    # 浮点乘
        STA PRODUCT       # 存储乘积
        HPR

# 浮点数据 (IBM 704 格式)
# 12000.0 = 符号 0, 指数 141 (128+13), 尾数...
MULTIPLICAND  OCT 413600000000  # 12000.0
MULTIPLIER    OCT 413600000000  # 12000.0
PRODUCT       OCT 0             # 结果

        END START
```

### 为什么 IBM 704 是 LEGENDARY？

1. **最古老的可行架构** - 1954 年，比任何其他可模拟的计算机都早
2. **浮点运算先驱** - 第一台硬件浮点计算机
3. **磁芯内存首创** - 第一台使用磁芯内存的量产计算机
4. **编程语言摇篮** - FORTRAN 和 LISP 的诞生地
5. **科学计算里程碑** - 用于 Sputnik 轨道计算、核研究等
