# IBM 704 历史背景

## 计算机史上的里程碑

**IBM 704** 于 1954 年推出，是计算机发展史上的重要里程碑。它不仅是 IBM 第一台量产的科学计算机，更是多个"第一"的创造者。

---

## 时代背景

### 1950 年代的计算机

```
┌─────────────────────────────────────────────────────────────┐
│  1950 年代计算机发展时间线                                   │
├─────────────────────────────────────────────────────────────┤
│  1951: UNIVAC I        - 第一台商用计算机                   │
│  1952: IBM 701         - IBM 第一台科学计算机               │
│  1953: IBM 650         - 第一台量产计算机                   │
│  1954: IBM 704         - 第一台浮点计算机 ⭐                 │
│  1956: IBM 305 RAMAC   - 第一台硬盘计算机                   │
│  1958: IBM 709         - IBM 704 的改进版                    │
│  1960: IBM 7090        - 晶体管版本                         │
└─────────────────────────────────────────────────────────────┘
```

### 技术环境

1950 年代是**真空管计算机的黄金时代**：

- **真空管**：计算机的基本构建块
- **磁鼓内存**：早期存储技术
- **打孔卡片**：主要输入/输出介质
- **机器语言**：直接二进制编程
- **批处理**：唯一的操作模式

2. **大容量内存** - 4096 字磁芯内存
3. **高可靠性** - 相比 IBM 701 改进
4. **易于编程** - 促成 FORTRAN 诞生
