# RustChain Miner - Native Rust

Port of RustChain miner to Rust with 7 hardware fingerprint checks.

## Features
- 7 hardware fingerprint checks (CPU, cache, clock, jitter, thermal, anti-emulation)
- Ed25519 attestation
- Cross-platform (Windows, macOS, Linux)

## Build
```bash
cargo build --release
```
